Dependencies needed for Open Cloth C++ projects are:

freeglut
glew
glm

Dependencies needed for the CUDA Open Cloth C++ project are:

CUDA
openCL

(note: As of January of 2020, I haven't tried downloading or installing or compiling with CUDA, so I can't say much here.  Only thing of note is that the CUDA dll that appears in the bin directory is version 3.0, so you may want to start with that version.)

Dependencies needed for Open Cloth WebGL projects are:

three.min.js
stats.min.js
